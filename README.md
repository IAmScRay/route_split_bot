# Инструкция по установке
1. Клонируем репозиторий и переходим в папку: `git clone https://github.com/IAmScRay/route_split_bot && cd route_split_bot`
2. Создаём виртуальную среду `venv` и активируем её: `python3 -m venv venv && source venv/bin/activate`
3. Устанавливаем нужные библиотеки: `pip install -r req.txt`
4. В `app.py` в начале нужно указать токен ТГ-бота и API-ключ от OpenAI [тут](https://platform.openai.com/), желательно - уже с балансом в пару баксов. Если хочется – можно поменять модель с `gpt-5.4-mini` на другую (нужно смотреть документацию по доступным моделям на платформе OpenAI).

## Запуск в фоновом режиме
1. Для запуска в фоне, создаём сессию `tmux` с желаемым названием: `tmux new -s <название>`. Сессия откроется в той же папке, в которой мы были.
2. Внутри открытой сессии активируем среду снова: `source venv/bin/activate`.
3. Запускаем: `python3 app.py`

Для выхода в основной терминал нажимаем комбинацию `Ctrl + B`, затем быстро нажимаем `D`.

Для возврата в терминал с программой, подключаемся к сессии по имени: `tmux attach -t <названиие>`.
